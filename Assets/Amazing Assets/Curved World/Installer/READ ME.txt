Curved World asset contains 3 packages for each Unity render pipeline:

1) Built-in - package for built-in (default) render pipeline.
2) Universal - package for Universal scriptable render pipeline (URP).
2) High Definition - package for High Definition scriptable render pipeline (HDRP).

Based on the project's render pipeline import one of the provided package.


------------------------------------------------------------------
Curved World v2021.2 requires Unity version 2019.4.26 and SRP v7.6
